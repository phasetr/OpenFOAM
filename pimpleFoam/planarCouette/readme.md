# 2 次元クエット流
## 参考
- [P.123, 第4章 粘性流体の力学](http://www.esst.kyushu-u.ac.jp/textbook/chap04.pdf)

> 2 枚の平行な板に挟まれた密度一様流体がある．
> 下の板は静止し，上の板は速度Uでx軸方向に動いているとすると，
> その間の流体がどのように運動するか考えよう．
> (中略)
> 両板の間の流体の流速は直線分布になることがわかる．
> このような流れをクエット流(Couette flow)と呼ぶ．
