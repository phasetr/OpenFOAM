# 2 次元ポアズイユ流
## 参考
- [P.123, 第4章 粘性流体の力学](http://www.esst.kyushu-u.ac.jp/textbook/chap04.pdf)

> 静止している2枚の平行平板で挟まれた流体が一定の圧力勾配(∂p/∂x=\0)で流れる場合を考えると，
> 流速は (中略) のように放物線分布となる．
> この流れを平面ポアズイユ流(plane Poiseuilleflow)と呼ぶ．
